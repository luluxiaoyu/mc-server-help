# 服务器地址端口隐藏

提示

你需要准备一个域名！ 没域名的请看第一步

# 购买域名

推荐使用阿里云：https://wanwang.aliyun.com/domain/

（我没用我自己的推广链接！！！）

如果你是第一次接触域名-你需要准备的材料

身份证号码（年龄不限，学生也是可以的）
支付宝（用于阿里云付款）

把想注册的域名加入清单，结算

然后按照提示付款，实名验证即可

# 域名解析

进入域名解析控制台（阿里云作为示例）

添加记录，选择srv记录

主机记录填写：

_minecraft._tcp.子域名

记录值填写：

5 0 源地址端口 源地址域名/IP

看不懂什么意思？请看下面的例子：选择1或选择2

# 选择1-解析到子域名

现在假如我的服务器（内网穿透）地址是xxx.cn:12345

假如我的域名是xx.com,我想把我服务器地址改为mc.xx.com

那么，你应该填写： 主机记录填写：

_minecraft._tcp.mc

记录值填写：

5 0 12345 xxx.cn

# 选择2-解析根域名

现在假如我的服务器（内网穿透）地址是xxx.cn:12345

假如我的域名是xx.com,我想把我服务器地址改为xx.com

那么，你应该填写： 主机记录填写：

_minecraft._tcp

记录值填写：

5 0 12345 xxx.cn
