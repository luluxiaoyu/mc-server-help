# 快速开始

# 1-准备Java环境

提示

如果你已经有Java环境，请跳过这一步！

点击开服器的下载选项卡-安装Java

安装后，请重启电脑以使环境变量生效！

# 2-下载服务端

选择你喜欢的服务端，双击下载

原版服务器推荐：spigot（内置的paper版本太旧了，不建议下载）

mod端推荐：mohist和cat端（其中mohist有大部分的日志汉化和游戏汉化）

然后点击开服

然后需要同意EULA

点击“读完”

然后，在弹出的记事本把eula=false改成true（不是“ture”！！！）

保存，然后重新开启服务端

看见“done”即可！

# 3-关闭正版验证

提示

有正版/服务器面向正版玩家的，这一步可以跳过！

进入设置-配置服务器

关闭正版验证，关掉窗口，然后重启服务器即可

# 4-内网穿透

提示

拉了企业宽带+公网ip的大佬，有动态IPV4的大佬，用云服务器的，只用IPV6的大佬可以绕过本步骤

点击内网穿透

选择离你最近的位置

然后输入服务端端口（默认25565），和你的QQ号码

创建映射，根据提示让其他人连接即可！
