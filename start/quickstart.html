# 1-准备Java环境

提示

如果你已经有Java环境，请跳过这一步！

# 如何选择java？

1.8-1.13 只能使用java8

1.14-1.16 可以使用java8-java15 （1.14和1.15并没有测试过，推荐java8）

1.17 你必须使用java16

# 下载java-仅提供Windows版本

下载源使用-清华大学开源软件镜像站和bmclapi

Java8 64位 (opens new window)

Java11 64位 (opens new window)

Java15 64位 (opens new window)

Java16 64位 (opens new window)

下载后，运行安装即可，Java11，15，16记得勾选java_path！

# 2-下载服务端

选择你喜欢的服务端，双击下载

原版服务器推荐：spigot或者paper及其分支

mod端推荐：mohist和cat端（其中mohist有大部分的日志汉化和游戏汉化）

然后点击开服

然后需要同意EULA

"好好阅读一下"

看见“done”即可！

# 3-关闭正版验证

提示

有正版/服务器面向正版玩家的，这一步可以跳过！

::: warn 警告 下面部分针对开服器3.3版本

新版开服器尚未更新

请自行修改server文件

:::

进入设置-配置服务器

关闭正版验证，关掉窗口，然后重启服务器即可

# 4-内网穿透

提示

拉了企业宽带+公网ip的大佬，有动态IPV4的大佬，用云服务器的，只用IPV6的大佬可以绕过本步骤

点击内网穿透

选择离你最近的位置

然后输入服务端端口（默认25565），和你的QQ号码

再次点击“内网穿透”，出现如图文字即为成功
