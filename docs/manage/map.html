    # 地图管理

    提示

    本页文档基于Spigot为核心的服务器（也就是spigot和paper）
    其它一不一样，不知道

    # 默认格式

    为了方便多世界

    spigot将各个维度分开存放

    服务端文件夹中，这三个文件夹即为地图文件

    # 备份

    推荐插件：ebackup

    每天定时可以备份你的地图及服务器文件

    链接：https://www.mcbbs.net/thread-957949-1-1.html

    # 多世界

    请参考MCBBS中的帖子

    多世界MV插件：https://www.mcbbs.net/thread-1016455-1-1.html

    多世界传送门：https://www.mcbbs.net/thread-584063-1-1.html
